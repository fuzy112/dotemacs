# Dotemacs

This is my personal Emacs configuration. It is only for my personal use, but you may be interested in some packages I wrote:

| Package                                                        | Description                                                |
|----------------------------------------------------------------|------------------------------------------------------------|
| [`bangs.el`](user-lisp/bangs.el)                               | Kagi bangs for Emacs                                       |
| [`rust-docs.el`](user-lisp/rust-docs.el)                       | Look up Rust documentation                                 |
| [`flymake-define.el`](user-lisp/flymake-define.el)             | Convenient macro for defining Flymake diagnostic functions |
| [`good-doc.el`](user-lisp/good-doc.el)                         | View Gtk documentation in Emacs                            |
| [`hide-pass.el`](user-lisp/hide-pass.el)                       | Hide passwords, API keys, tokens from the buffer           |
| [`consult-nixos.el`](user-lisp/consult-nixos.el)               | Look up NixOS options                                      |
| [`consult-ugrep.el`](user-lisp/consult-ugrep.el)               | Run ugrep like consult-ripgrep                             |
| [`consult-kill.el`](user-lisp/consult-kill.el)                 | Search and kill process using Consult interface            |
| [`consult-browser-hist.el`](user-lisp/consult-browser-hist.el) | Search browser histories with Consult interface            |
| [`tencent-cos.el`](user-lisp/tencent-cos.el)                   | Upload files to Tencent Cloud COS                          |
